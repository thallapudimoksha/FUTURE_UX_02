# Mobile App UI for Appointment Booking System

## Project Overview
This project focuses on designing a mobile application UI for an appointment booking system. The application helps users book appointments easily for services such as salons, clinics, fitness trainers, and consultants.

## Objective
To create a user-friendly and mobile-first appointment booking experience that is simple, efficient, and easy to navigate.

## Features
- User Login / Signup
- Service Selection
- Date & Time Slot Booking
- Appointment Confirmation
- Clean and Responsive UI Design

## Tools Used
- Canva
- UI/UX Design Principles

## Screens Included
1. Login Screen
2. Service Selection Screen
3. Appointment Booking Screen
4. Confirmation Screen

## Outcome
Designed a simple and intuitive mobile application interface that improves the appointment booking experience for users.
